Tags de Cabeçalho: <h1> a <h6>

Essas tags são usadas para definir cabeçalhos de diferentes níveis de importância.

Exemplo:

Título Principal - <h1>

Subtítulo - <h2>

Título Menor - <h3>

Título Secundário - <h4>

Título Pequeno - <h5>
Menor Título - <h6>

Tag de Parágrafo: <p>

Esta tag é utilizada para definir um parágrafo de texto.

Tag de Marcação: <mark>

Utilize esta tag para destacar texto, como se fosse um marca-texto.

Tag de Texto Pequeno: <small>

Esta tag exibe texto em tamanho reduzido. Exemplo de texto pequeno.

Tags de Estilo: <i>, <u>, <strong>

Use italico com a tag <i>, sublinhado com a tag <u> e negrito com a tag <strong>.

Listas Ordenadas e Não Ordenadas: <ol>, <ul>, <li>

Exemplo de Lista Ordenada:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Exemplo de Lista Não Ordenada:

  • Item A
  • Item B
  • Item C

Link: <a>

Use a tag <a> para criar links. Exemplo: Clique aqui para visitar o site da DIO.

Citação: <blockquote>

"Esta é uma citação famosa." - Alguém Famoso

Texto Riscado: <del>

Esta tag exibe texto riscado.

Subscrito e Sobrescrito: <sub>, <sup>

Use a tag <sub> para subscrito, como em H2O.

Use a tag <sup> para sobrescrito, como em E = mc2.

Abreviações: <abbr>

Use a tag HTML para definir abreviações.

Dicas de boas práticas:

  • Mantenha o código HTML organizado e bem indentado.
  • Use tags semânticas para melhorar a acessibilidade e o SEO.
  • Evite estilos embutidos; prefira usar arquivos CSS externos.
  • Valide o HTML para garantir que ele esteja livre de erros.
  • Utilize um tema de cores que seja agradável para leitura e que atenda aos critérios de acessibilidade.
